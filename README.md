# Social Media Addiction Classification

## Project Overview
This project predicts whether a user is addicted to social media using behavioral indicators.

## Problem Type
Binary Classification

## Models Used
- Logistic Regression
- Random Forest Classifier

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score

## Key Insights
- High correlation between behavioral indicators and addiction.
- Model achieved ~99% accuracy.
- Robustness validated by removing highly correlated features.

## Tools & Libraries
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
